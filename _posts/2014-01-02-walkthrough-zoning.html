Walkthrough: The "Zoning" Area

Zoning Districts reflect your communityâ€™s Zoning Ordnance. Zoning determines whether businesses within each Use Code are appropriate at any given location.

As most of your staff knows, within each Zoning District, each Use Code is either Permitted (outright or conditionally) or Prohibited. When launched for your organization, the OpenCounter system was configured to match the existing zoning ordinance precisely. Thus, changes to the zoning districts in the OpenCounter system should probably only follow changes to the legally binding zoning ordinance itself.

Each Zoning District also has the following attributes:

  • An Acronym Code
  • A Classification
  • A Name
  • A Description
  • Permissions

You can edit any of these attributes at your discretion, simply by clicking the "edit" button and reviewing the information shown in the screenshot below.
