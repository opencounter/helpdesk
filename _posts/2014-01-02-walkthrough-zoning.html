Admin Walkthrough: The "Zoning" Area

Zoning Districts reflect your communityâ€™s Zoning Ordnance. Zoning determines whether businesses within each Use Code are appropriate at any given location.

These determinations are called Permissions. Within each Zoning District, each Use Code is either Permitted or Prohibited.

Each Zoning District also has the following attributes:

  • An Acronym Code
  • A Classification
  • A Name
  • A Description
  • Permissions
